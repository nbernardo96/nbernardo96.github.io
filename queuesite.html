Tony's Queue

Role: Fullstack Developer
Technologies used: Python, Django (Backend), React (Frontend)

Tony's Queue is a website created for clients to request or place an order for customized business cards and flyers.

My goal for this project was to create a minimalistic and simple website. Frontend-wise, I helped design and code the home, business card form, and flyer form pages. For the backend, my team and I decided to use Django, and PostgreSQL for the database. I worked on making the necessary tables and columns for the database, and integrated it with the frontend.
